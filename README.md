# flutter_study_demo

这是我的一个flutter入门练习项目,适配Android和iOS。

## 主要功能

- 仿写实现首页、搜索、旅拍模块；
- 旅拍模块实现瀑布流布局；
- flutter基础组件的练习使用；
- 全局设置语言国际化；
- 全局设置主题颜色样式；
- 手动自动设置白天/暗黑模式；
